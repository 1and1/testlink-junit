  • public class TestLinkLoggingRunListener
extends AbstractTestLinkRunListener<net.oneandone.testlinkjunit.tljunit.InTestLinkLogRunListener>
    This class logs the start and end of executed tests to the injected Logger.
    • The name of the Logger may be set by means of the system property testlink.loggername, which defaults to "TESTLINK".
    • A system property called testlink.uri will be used to log a direct link to the test case in your Testlink instance. When unavailable or empty will suppress the link.
    For more usage information, see TestLinkXmlRunListener.
